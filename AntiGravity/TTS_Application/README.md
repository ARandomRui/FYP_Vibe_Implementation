# Text-to-Speech Application (AntiGravity)

Welcome to the AntiGravity implementation of the Text-to-Speech (TTS) Application!

This is a native Android application.

## How to Run Locally

1. **Prerequisites**: Ensure you have [Android Studio](https://developer.android.com/studio) installed on your computer.
2. **Open the Project**:
   - Launch Android Studio.
   - Click on **Open**.
   - Navigate to the `AntigravityAndroidTTS` directory in this repository and select it.
3. **Sync Gradle**: Allow Android Studio a few moments to download dependencies and sync the Gradle project.
4. **Run the App**: 
   - Connect a physical Android device via USB (with USB Debugging enabled) or start an Android Emulator from the AVD Manager.
   - Click the green **Run (▶)** button in the top toolbar to build and deploy the app.
