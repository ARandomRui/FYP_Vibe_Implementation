# Chrome Dinosaur Game (AntiGravity)

Welcome to the Chrome Dinosaur Game! 

This is a Python-based implementation of the classic Chrome Dinosaur game using the Pygame library.

Note: Due to Antigravity's Update from 1.0 to 2.0, this project encountered an error when trying to fetch past chat, this is the only project that encountered such error. 

## How to Run Locally

1. **Prerequisites**: Ensure you have Python installed on your system.
2. **Install Pygame**: Open your terminal and install the required library:
   ```bash
   pip install pygame
   ```
3. **Navigate to the Source Code**:
   ```bash
   cd "Google Dinosaur Game"
   ```
4. **Run the Game**:
   ```bash
   python main.py
   ```
   *(Use `python3` instead of `python` if you are on macOS/Linux)*
