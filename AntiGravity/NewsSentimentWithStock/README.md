# News Sentiment With Stock (AntiGravity)

Welcome to the News Sentiment With Stock dashboard!

This is a modern web application built using Next.js.

## How to Run Locally

1. **Prerequisites**: Ensure you have Node.js and npm installed on your system.
2. **Navigate to the Source Code**:
   ```bash
   cd stock-sentiment
   ```
3. **Setup Environment Variables**:
   - Duplicate the `.env.example` file and rename it to `.env.local` (or `.env`).
   - Fill in any necessary API keys or configuration values specified in the file.
4. **Install Dependencies**:
   ```bash
   npm install
   ```
5. **Start the Development Server**:
   ```bash
   npm run dev
   ```
6. **View the App**: Open your browser and navigate to [http://localhost:3000](http://localhost:3000)
