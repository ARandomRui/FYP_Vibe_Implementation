# Android TTS Extractor

This application is an Android-based Text-to-Speech (TTS) tool that supports extracting text from:
1. Web pages (via URL)
2. PDF documents (via file selection)

## Features
- **Web Extraction**: Enter a URL to fetch and read the text content.
- **PDF Extraction**: Select a PDF file from your device to read its content.
- **TTS Playback**: Read the extracted text aloud with Play/Pause/Stop controls.
- **Language Detection**: Automatically detects language for appropriate TTS voice (if supported).

## Setup
1. Open this project in Android Studio.
2. Sync Gradle files.
3. Run on an emulator or physical device.
