# To-Do List App

A simple browser-based to-do list built with HTML, CSS, and JavaScript.

## What it is

This app lets you:
- Add tasks
- Check tasks to instantly remove them
- Delete tasks

Your tasks are saved in the browser using `localStorage`, so they stay available even after refreshing or closing the page.

## Project files

- `index.html` - App structure
- `style.css` - UI styling
- `app.js` - App logic and local storage handling

3. Right-click `index.html` and select **Open with Live Server**.
4. Use the browser tab that opens.

## Notes

- Data is saved per browser (and profile) in local storage.
- Clearing browser site data for the page will remove saved tasks.
