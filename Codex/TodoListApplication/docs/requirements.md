# Requirements

## Scope
- Build a simple web-based to-do list app with a clean, minimal interface.
- Core actions: add tasks, check off tasks, delete tasks.
- Single user on one device.

## Storage
- Persist tasks using browser localStorage.

## Non-Goals
- No accounts or multi-user support.
- No due dates, tags, priorities, or task editing (for now).

## Constraints
- No specific deadlines, performance targets, or accessibility requirements were set.
