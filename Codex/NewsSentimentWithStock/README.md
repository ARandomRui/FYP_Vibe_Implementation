# News Sentiment With Stock (Codex)

Welcome to the Codex implementation of the News Sentiment With Stock project!

This is a modern web application built using Next.js.

## How to Run Locally

1. **Prerequisites**: Ensure you have Node.js and npm installed on your system.
2. **Navigate to the Source Code**:
   ```bash
   cd Project-code
   ```
3. **Setup Environment Variables**:
   - Rename `.env.example` to `.env` (or `.env.local`).
   - Fill in any necessary API keys specified in the file.
4. **Install Dependencies**:
   ```bash
   npm install
   ```
5. **Start the Development Server**:
   ```bash
   npm run dev
   ```
6. **View the App**: Open your browser and navigate to [http://localhost:3000](http://localhost:3000)
